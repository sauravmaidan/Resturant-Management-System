# Restaurant Management System (Customer View)

## Description
This is a simple Restaurant Management System designed only for the **customer view**.  
Customers can view the menu, search dishes, place orders, generate bills, and add reviews.

## Features
- View full menu
- Search dishes by name
- Place order for selected dishes
- Generate bill for the order
- Add review for the restaurant or dish

## How to Use
1. Open the project
2. Run the main file
3. Choose options from the menu:
   - View Menu
   - Search Dish
   - Order
   - Generate Bill
   - Add Review
